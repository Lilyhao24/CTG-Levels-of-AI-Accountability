# Changelog

All notable changes to the CTG Model are documented here.

## [v3.2] - 2026-03-14 (Current)

### Changed
- **C-axis renamed** from "Cognitive Continuity" to "State Continuity" — resolves confusion between intelligence and state persistence
- **DDI formalized** with Behavior Envelope baseline — DDI = deviation from expected behavior envelope, not raw behavioral change
- **T2 clarified** — explicitly requires "decision justification interface", not "explainable model"
- **G3 split** into G3a (regulator-assigned auditor) and G3b (accredited pool self-selection) for political adaptability

### Added
- Behavior Envelope concept for DDI (§3.3)
- DDI breaker clarification: "pause for human review, not terminate evolution" (§3.3)
- Grey-box ↔ T3 change log relationship clarification (§3.4)
- Digital Trust model as Open Question #7
- Behavior Envelope calibration as Open Question #8

## [v3.1] - 2026-03-14

### Added
- Standard Contestor Profile for T2 evaluation stability (§3.2)
- Multi-metric Evaluation Set replacing single "contestation success rate" (§3.2)
- DDI (Decision Drift Index) replacing "cognitive entropy" terminology (§3.3)
- Grey-box Preservation mechanism for breaker-moment evidence (§3.4)
- T3 cost feasibility statement — T3 only for high-risk domains (§3.5)
- EU AI Act mapping table (Chapter 6)
- Category Presumption for G2 user responsibility (§4.2)
- Layered Audit for G3 (compliance layer + domain expert layer) (§4.4)
- 2D projection visualization framework (§1.3)
- Anti-Label-Illusion mechanism with example text (§4.3)
- Specific first-landing scenarios: financial AI, medical AI, autonomous driving (§8.2)

### Changed
- C3 split into C3a (online learning, already exists) and C3b (auto-telic, theoretical)
- Original A2 "Explainability" → T2 "Contestability" paradigm shift
- I-axis (Memory Integrity) merged into T-axis as change traceability sub-dimension

## [v3.0] - 2026-03-14

### Changed
- **Architecture: dual-axis → three-axis (C → T → G)**
- C (Cognitive Continuity) — AI's own state persistence
- T (Traceability Infrastructure) — external observation & contestation capability  
- G (Governance Accountability) — institutional accountability framework
- G-axis subject explicitly declared as "institutions and people, not AI"
- Three axes have directed dependency (not orthogonal): C → T → G
